## About the Project

This website features a multi-page structure showcasing a commercial renovation project:

- **Homepage** - Hero section with project overview and feature cards
- **Ocio** - Entertainment and leisure offerings (cinema, arcade, bowling)
- **Gastronomía** - Dining and restaurant options
- **El Proyecto** - Project details with architectural renders
- **Contacto** - Multi-profile contact form for different audiences

The entire site is built without frameworks or build tools, demonstrating modern web design with pure HTML and CSS.
